## [1.0.0] - 2026-03-06

#### Added (새 기능)

**NLP 자연어 처리**

## 마이그레이션 가이드

### v0.9.x → v1.0.0

#### Breaking Changes

1. **API 응답 형식 변경**
   ```
   v0.9: { data: {...} }
   v1.0: { success: true, data: {...}, meta: {...} }
   ```

2. **인증 방식 변경**
   ```
   v0.9: Basic Auth
   v1.0: JWT Bearer Token
   ```

3. **데이터베이스 마이그레이션**
   ```bash
   npm run migrate:up
   ```

4. **환경변수 추가**
   ```
   TENANT_ID=default
   AUTH_MODE=jwt
   ```

#### 자동 마이그레이션

```bash
npm run migrate:v0.9-to-v1.0
```
