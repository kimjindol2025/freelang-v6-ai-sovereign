# 📋 Contributor Covenant Code of Conduct

## 우리의 약속

FreeLang 커뮤니티의 일원으로서, 우리는 모든 사람을 위해 열려있고 환영하는 환경을 만들기로 약속합니다. 나이, 신체 크기, 장애, 민족성, 성 정체성, 경험 수준, 국적, 외모, 인종, 종교, 성 정체성 또는 성적 지향에 관계없이 모두를 존중합니다.

---

## 우리의 기준

긍정적인 환경을 만드는 데 기여하는 행동:

✅ **환영하는 언어 사용**
- 모두를 포용하는 표현 사용
- 다양성을 존중하는 태도

✅ **존중과 공감**
- 다른 관점과 경험을 소중히 여김
- 건설적인 비판만 제공

✅ **책임감 있는 행동**
- 실수를 인정하고 영향을 받은 사람들로부터 배우기
- 개선에 집중

✅ **커뮤니티 중심**
- 커뮤니티 전체의 이익을 먼저 고려
- 아이디어와 지식을 나누기

---

## 용인하지 않는 행동

❌ **차별과 괴롭힘**
- 성적 언어나 이미지 사용
- 의도적인 괴롭힘
- 신원 또는 신념에 대한 공격

❌ **폭력과 위협**
- 폭력적인 언어나 위협
- 개인 정보 공개 (도싱)
- 안전에 대한 위협

❌ **남용적 행동**
- 지속적인 방해
- 부적절한 댓글
- 비상호적 성적 관심

❌ **출처 왜곡**
- 악의적 거짓 정보
- 신뢰 기반 파괴

---

## 시행

### 신고 방법

부적절한 행동을 목격하면:

1. **직접 대면** (안전한 경우):
   - 정중하게 중단을 요청
   - 개인 정보는 공개하지 않기

2. **이메일 신고**:
   - conduct@freelang.ai로 보내기
   - 구체적인 내용 포함
   - 증거 자료 첨부

3. **GitHub 신고**:
   - Repository admins 멘션
   - 상세한 설명 제공

### 신고 포함 내용

- 부적절한 행동의 설명
- 발생 시간과 장소
- 관련자 정보
- 증거 (메시지, 스크린샷)
- 영향을 받은 방식

### 익명성 보장

신고자의 신원은 요청 시 비공개로 유지됩니다.

---

## 시행 지침

### 조치 단계

1️⃣ **경고**
- 경미한 위반
- 첫 번째 사건
- 비의도적 위반

2️⃣ **일시 제한**
- 심각한 위반
- 반복된 경고 무시
- 위험한 행동

3️⃣ **영구 제외**
- 극심한 또는 지속적 위반
- 위협이나 폭력
- 신체적/정신적 안전 위험

### 항소 프로세스

처벌에 동의하지 않으면:

1. appeal@freelang.ai로 항소 제출
2. 다른 관리자가 검토
3. 최종 결정 14일 이내 통보

---

## 리더십 책임

프로젝트 리더는:

✅ **명확한 기준 설정**
- 행동 규범 공개
- 정기적 검토 및 업데이트

✅ **공정한 시행**
- 증거에 기반한 결정
- 편견 없는 처벌

✅ **커뮤니티 보호**
- 신고자 보호
- 피해자 지원

✅ **투명성 유지**
- 결정 이유 설명
- 절차 공개

---

## 커뮤니티 참여 권리

모든 커뮤니티 멤버는:

✅ 존중과 존엄성으로 대우받을 권리
✅ 안전한 환경에서 참여할 권리
✅ 피해 발생 시 보호받을 권리
✅ 공정한 절차에 따를 권리
✅ 자신의 의견을 표현할 권리

---

## 적용 범위

이 규범은 다음에 적용됩니다:

- GitHub 저장소 (이슈, PR, 토론)
- 공식 이메일 (conduct@freelang.ai)
- Slack/Discord 채널
- 컨퍼런스 및 행사
- 공식 소셜 미디어
- 프로젝트를 대표하는 모든 공간

**사적 의사소통은 제외되지만**, 프로젝트 참여가 중단될 수 있습니다.

당신의 관점도 소중합니다. 함께 배워봅시다."

"실수했네요. 지적해주셔서 감사합니다.
다음부터 주의하겠습니다."
```

```

```

```

```
"당신의 종교 때문에 당신을 신뢰할 수 없어."
```
