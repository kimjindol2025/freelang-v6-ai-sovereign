### 1. json-transform.json
**목적**: 배열 데이터 변환

**학습 포인트**: Array.map, String.upper 활용

```
입력: ["alice", "bob", "charlie"]
출력: ["ALICE", "BOB", "CHARLIE"]
```

### 2. text-analysis.json
**목적**: 텍스트 분석
**기능**: 문자열을 공백 기준으로 분할하고 단어 수 계산
**학습 포인트**: String.split, Array.length

```
입력: "Hello World Hello AI Hello CLAUDELang"
출력: 8개 단어
```

### 3. conditional-loop.json
**목적**: 조건문과 반복문 결합
**기능**: 1-5 숫자 중 홀수와 짝수 분류
**학습 포인트**: Loop + Condition 중첩

```
입력: [1,2,3,4,5]
출력: 홀수와 짝수 분류
```

### 4. reduce-example.json
**목적**: 배열 감축 (합계)
**기능**: 배열의 모든 수를 더하기
**학습 포인트**: Array.reduce, Lambda functions

```
입력: [10, 20, 30, 40]
출력: 100
```

### 5. nested-data.json
**목적**: 중첩된 배열 처리
**기능**: 2D 배열 각 행의 합계 계산
**학습 포인트**: 중첩 배열, Array<Array<T>>

```
입력: [[1,2,3], [4,5,6]]
출력: [6, 15]
```

### 6. string-processing.json
**목적**: 문자열 처리
**기능**: 문자열을 단어로 분할하여 대문자로 변환 후 조합
**학습 포인트**: String 체인 연산

```
입력: "the quick brown fox"

```

### 7. multiple-transforms.json
**목적**: 다중 변환 체이닝
**기능**: 필터 → 매핑 → 최종 결과
**학습 포인트**: 함수형 프로그래밍 패턴

```
입력: [1,2,3,4,5]
필터: x > 1
변환: x * 10
출력: [20, 30, 40, 50]
```

### 8. data-validation.json
**목적**: 데이터 검증

```

```


### 10. data-grouping.json

**학습 포인트**: 필터링 및 분류

```

```
